# python-onnx 开发网络服务

* 安装Onnx库
* 模型转换为Onnx格式
* Onnx模型的检验
* 使用Onnx Runtime进行推理

```py
pip install onnx 
# 安装onnx runtime
pip install onnxruntime # 使用CPU进行推理
# pip install onnxruntime-gpu # 使用GPU进行推理
```
